# 🖥️ The-Gaslight-OS
> "It's not a bug, it's a feature. It's not a crash, it's a conversation."

**The-Gaslight-OS** is a psychological prank suite designed to convince users that their computer has developed a sentient (and very annoying) personality. Built with **Python**, **Tkinter**, and custom **GIMP** designs.

---

## 🛠️ The Suite
This repository contains multiple modules designed for maximum digital confusion:

### 1. Sentient BSOD Simulator (`BSOD.py`)
A full-screen, high-priority overlay that mimics a system crash but with a twist.
* **System Lockdown:** Hooks into the keyboard layer to block `Win Key`, `Alt+Tab`, `Esc`, and `Delete`.
* **Vibration Mode:** Custom error screens warning that your hardware is physically attempting to escape.
* **Linux Identity Crisis:** Triggers a fake kernel panic where the PC forgets it's running Windows.
* **Invisible Mouse:** Disables the cursor to remove the user's sense of control.

### 2. The Infinite BTC Grift (`idk.py`)
A social engineering script that promises wealth but delivers only disappointment.
* **Fake Wallet Generation:** Creates "100 BTC Private Key" files in random user directories.
* **The Gaslight Loop:** Tells the user to delete the file to claim the reward, then mocks them when they run it again.
* **Vibe Check:** Uses "AI heuristics" to tell the user they don't deserve the money.
