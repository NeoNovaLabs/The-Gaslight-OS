# 🖥️ The-Gaslight-OS
> "It's not a bug, it's a feature. It's not a crash, it's a conversation."

**The-Gaslight-OS** is a psychological BSOD (Blue Screen of Death) simulator designed to convince the user that their computer has developed a personality—and that personality is having a very bad day. Built with **Python**, **Tkinter**, and custom **GIMP** designs.

---

## 🛠️ Features
* **System Lockdown:** Hooks into the Windows keyboard layer to block `Win Key`, `Alt+Tab`, `Esc`, and `Task Manager`.
* **Sentient Error Screens:** A gallery of custom-made nonsense errors, including:
    * **Vibration Mode:** Warnings that your monitor might walk off the desk.
    * **Linux Identity Crisis:** The PC forgets it's running Windows.
* **Invisible Mouse:** Hides the cursor for a truly "locked" feeling.

---

## ⌨️ Controls
| Key | Action |
| :--- | :--- |
| **Arrow Right / Left** | Cycle through the BSOD gallery |
| **Enter** | Enter "Full Prank" mode (Fullscreen + Hidden Mouse) |
| **Ctrl + Alt + A + B** | **SECRET EXIT:** Terminate the script and unblock keys |
